import { Bar, BarChart, XAxis, YAxis, CartesianGrid, Tooltip, Legend } from 'recharts';

## Introduction

This blog post serves as a demonstration of how you can now include both standard markdown tables and interactive charts within your content using MDX.

## Sample Table

Here is an example of a standard markdown table:

| Header 1 | Header 2 | Header 3 |
|---|---|---|
| Data A1 | Data B1 | Data C1 |
| Data A2 | Data B2 | Data C2 |
| Data A3 | Data B3 | Data C3 |

This table is created using the simple markdown syntax you are familiar with.

## Sample Chart

Below is an example of a bar chart embedded using the `ChartContainer` component.

<ChartContainer
  config={{
    sales: {
      label: 'Sales',
      color: 'hsl(var(--chart-1))',
    },
    revenue: {
      label: 'Revenue',
      color: 'hsl(var(--chart-2))',
    },
  }}
  className="min-h-[300px] w-full"
>
  <BarChart
    accessibilityLayer
    data={[
      { month: 'Jan', sales: 150, revenue: 200 },
      { month: 'Feb', sales: 200, revenue: 250 },
      { month: 'Mar', sales: 180, revenue: 220 },
      { month: 'Apr', sales: 250, revenue: 300 },
      { month: 'May', sales: 220, revenue: 280 },
      { month: 'Jun', sales: 280, revenue: 350 },
    ]}
  >
    <CartesianGrid vertical={false} />
    <XAxis dataKey="month" axisLine={false} tickLine={false} tickMargin={8} />
    <YAxis axisLine={false} tickLine={false} tickMargin={8} />
    <Tooltip cursor={false} content={<Tooltip />} />
    <Bar dataKey="sales" fill="var(--color-sales)" radius={4} />
    <Bar dataKey="revenue" fill="var(--color-revenue)" radius={4} />
  </BarChart>
</ChartContainer>

This chart is rendered using the Shadcn UI Chart component, which is built on top of Recharts. You can customize the chart type, data, and appearance by modifying the `config` and `data` props, and by using different components from the `recharts` library within the `ChartContainer`.
